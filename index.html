Simple maze generator

Repo

Description

This is a small program that can generate a random maze, solve it and display the maze and solution in the command prompt. It was written over a weekend.

This project illustrates my organization of a C project. Abstract data types are defined such as the Stack (pointer stack) and the Valstack (value stack). They have a defined interface which operates on a opaque pointer. This means that the data is hidden away, and the code is more modular.
